\# FTX Stablecoin Run: On-Chain Anatomy



Replication code for an empirical descriptive paper on stablecoin flows from the

FTX Hot Wallet during the November 2022 collapse.

\## Data sources



All data is publicly available and free:



\- \*\*Binance Vision\*\* (https://data.binance.vision/) - aggTrades for BTC/USDT spot,

&#x20; Oct 15 - Nov 25, 2022. \~3.5 GB.

\- \*\*Etherscan API V2\*\* (https://docs.etherscan.io/) - ERC-20 transfers for

&#x20; publicly-tagged FTX wallet addresses, USDT and USDC.
